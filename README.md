# Learn to Code from Scratch (Python)

A beginner-friendly, hands-on introduction to programming using Python.

## How to Use

Run any lesson file with Python to see the output and learn by example:

```bash
python lessons/01_variables.py
```

## Lessons

Work through these in order — each one builds on the previous:

| # | File | Topic |
|---|------|-------|
| 1 | `lessons/01_variables.py` | Variables, data types, math, strings |
| 2 | `lessons/02_conditionals.py` | If/elif/else, comparisons, and/or/not |
| 3 | `lessons/03_loops.py` | For loops, while loops, lists, break/continue |
| 4 | `lessons/04_functions.py` | Defining functions, parameters, return values |

## Exercises

Once you've gone through the lessons, test your skills:

```bash
python exercises/exercises.py
```

The exercises file has 5 challenges with solutions at the bottom. Try solving each one before looking at the answer!

## Tips for Beginners

1. **Type the code yourself** — don't just read it. Muscle memory matters.
2. **Experiment** — change values, break things, see what happens.
3. **Read error messages** — they tell you what went wrong and where.
4. **Go slow** — understanding one concept well beats rushing through many.
